# Brasa documentation

Welcome! This is the oficial documentation for Brasa — a Programming Language with **Portuguese Syntax** for native portuguese who are learning how to code.

Read the sections below for a quick guide of where to begin.

## Installation

To install Brasa and begin coding right away, follow the steps below:

1. Install [Python](https://www.python.org/downloads/).

2. Install Brasa package

```
$ pip install brasa
```

3. Run the following command

```
$ brasa
```

If you see the following messagem and a prompt, that means Brasa was successfully installed in your machine.

```
message
```

🎉🎉 Congratulations! Now that you have Brasa ready for use. You can read the section below and choose however you want to learn it.
